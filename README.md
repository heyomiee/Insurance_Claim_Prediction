# 🛡️ Insurance Claim Prediction

![Project Preview](home.png)

## 📌 Overview

Insurance Claim Prediction is a Machine Learning web application built using **Python** and **Streamlit**. It predicts estimated insurance charges based on user inputs such as age, BMI, smoking status, number of children, sex, and region.

---

## ✨ Features

- 🔮 Predict insurance charges instantly
- 🎨 Interactive and user-friendly Streamlit interface
- 🤖 Machine Learning-based prediction model
- 📊 Easy-to-use input form
- ⚡ Fast and accurate predictions

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
